# 02. Preliminary Coursework

The early assignments established the Sentaurus workflow before the pMOS optimization.

## Practice Topics

- Device structure visualization
- TDR output after process steps
- Transfer-curve comparison
- Gate length split
- Gate oxidation time split
- LDD dose split
- Source/Drain dose comparison
- SDevice Editor practice

## Key Observations

| Variable | Main Trend |
|---|---|
| Longer Lg | Lower Id and gm, improved short-channel stability |
| Longer GOxTime | Thicker oxide, higher Vtgm, lower Id and gm, higher SS |
| Higher LDD dose | Lower extension resistance and higher Id, with possible leakage penalty |
| Higher S/D dose | Small current increase with limited transfer-curve change |

These exercises were background work. The main portfolio result is the pMOS process conversion and optimization.

**Summary:**  
The preliminary work established the Sentaurus process, simulation, and parameter-analysis workflow.
